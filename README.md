# CV Multimédia

Ce projet contient un CV multimédia en HTML, CSS et JavaScript.

## Fonctionnalités
- Site bilingue Français / Anglais
- Vidéo synchronisée avec le parcours scolaire
- Audio synchronisé avec le parcours scolaire
- Pas de librairie externe utilisée

## Installation locale
1. Ouvrir le dossier `CV-multimedia-repo`
2. Ouvrir `index.html` dans un navigateur

## Git

```bash
git init

git commit -m "Initial commit"
git remote add origin https://github.com/<votre-utilisateur>/<votre-repo>.git
```
